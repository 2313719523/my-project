# 温度适配规则
[温度规则]
>30℃: 建议轻薄透气材质，如棉麻、真丝
20-30℃: 适合衬衫、薄外套、连衣裙
10-20℃: 适合毛衣、风衣、牛仔外套
0-10℃: 需要保暖，推荐羽绒服、厚毛衣
<0℃: 必须穿羽绒服、加绒裤、围巾手套

# 场合搭配规则
[商务场合]
正式会议: 西装套装、衬衫、皮鞋
商务晚宴: 西装、连衣裙、高跟鞋
日常办公: 衬衫、西裤、简约连衣裙

[休闲场合]
日常出行: T恤、牛仔裤、休闲鞋
朋友聚会: 卫衣、休闲裤、运动鞋
运动健身: 运动服、运动鞋、速干材质

[特殊场合]
约会: 连衣裙、小香风、高跟鞋
婚礼: 礼服、套装、精致配饰
面试: 正装、简洁大方、淡妆

# 身材适配建议
[梨型身材]
上衣: 选择V领、露肩上衣，吸引视线向上
下装: A字裙、阔腿裤，避免紧身裤
颜色: 上浅下深，平衡比例

[苹果型身材]
上衣: 选择深色、V领、有垂坠感的上衣
下装: 直筒裤、A字裙，突出腿部线条
避免: 高腰裤、紧身上衣

[沙漏型身材]
上衣: 合身不紧身，突出腰线
下装: 高腰裤、铅笔裙
重点: 突出腰部曲线

# 年龄风格建议
[20-25岁]
风格: 年轻时尚、活力四射
推荐: 卫衣、牛仔裤、连衣裙
颜色: 明亮色系、流行色

[25-30岁]
风格: 轻熟优雅、职场得体
推荐: 衬衫、西装、中长裙
颜色: 中性色、莫兰迪色系

# 新增场景规则
[婚礼宾客]
避免：白色（新娘色）、太暴露
推荐：及膝连衣裙、套装、精致配饰
颜色：淡雅色系、避免大红

[面试场合]
要点：专业、得体、不夸张
推荐：西装套装、衬衫、皮鞋
颜色：黑白灰、藏蓝、米色

[海边度假]
材质：透气、快干、防晒
推荐：吊带裙、短裤、草帽、太阳镜
颜色：亮色系、印花
